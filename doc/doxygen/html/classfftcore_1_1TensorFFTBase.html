Detailed Description

template<typename DataType, int Rank>
class fftcore::TensorFFTBase< DataType, Rank >

A wrapper for Eigen::Tensor.

TensorFFTBase behaves as a wrapper for Eigen::Tensor objects. It's meant to be used as it is or as a base class for type specific wrappers. This hides the use of Eigen tensors and allows the user to define their own cleaning and manipulation methods for input/output specific to their application through the extension of the TensorFFTBase class.

TimeTensor, FreqTensor
