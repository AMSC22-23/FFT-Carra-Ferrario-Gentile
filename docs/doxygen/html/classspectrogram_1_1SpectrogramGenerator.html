template<class FFTStrategy >
void spectrogram::SpectrogramGenerator< FFTStrategy >::compute_spectrogram ( bool decibel_scale = true,
bool assume_real = true )

Computes the spectrogram from the loaded signal. All the spectrograms are stored in a vector inside the generator.

Parameters
decibel_scaleIf true, the entries of the spectrogram are converted to decibels with the following transformation: x -> 10 * log10(x). The operation avoids numerical instability by adding a small epsilon to the argument of the logarithm. It defaults to false.
assume_realIf true, the spectrogram is computed assuming that the signal is real, thus reducing the number of frequency bins by half. It defaults to true.

Loads the signal from an existing tensor. The signal is divided into frames and stored inside the generator.

Parameters
signalThe signal to be transformed into a spectrogram.
frame_lenghtThe lenght of the frame window, in samples. It must be a power of 2.
frame_stepThe step between two consecutive frames, in samples. It default to frame_lenght.
pad_endIf false, the last frame is discarded if it lies partially outside the signal. If true, the last frame is padded with zeros to match the frame lenght.
