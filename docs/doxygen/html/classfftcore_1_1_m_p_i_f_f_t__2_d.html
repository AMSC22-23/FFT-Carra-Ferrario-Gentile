Detailed Description

template<typename FloatingType = double>
class fftcore::MPIFFT_2D< FloatingType >

MPI implementation of the 2 dimensional FFT.

MPIFFT_2D strategy simply executes the fft on every row of the input matrix, and then on every column, using one dimensional MPIFFT strategy.
