Detailed Description

template<typename FloatingType = double>
class fftcore::SequentialFFT< FloatingType >

Sequential implementation of the 1 dimensional FFT using Cooley-Tukey algorithm.

We provide a link to the <a href="https://en.wikipedia.org/wiki/Cooley%E2%80%93Tukey_FFT_algorithm" wikipedia page of the Cooley-Tukey algorithm for computing the FFT. This class provides a very basic implementation of the algorithm with no kind of optimization.
